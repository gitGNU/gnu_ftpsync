FTPSync.pl Homepage

General

FTPSync.pl is a PERL skript to synchronize a local directory tree and a remote FTP directory tree.

FTPSync.pl is also 's first open source project and is published under the GPL (GNU Public License).

FTPSync.pl was initially managed using the services of at ftpsync.sourceforge.net (project homepage) respectively sourceforge.net/projects/ftpsync/ ("internal" project page), but ...

From 1.3.00 (2009-04-15) on, FTPSync.pl got a subversion repository and trac page, all hosted at our own OpenSource site, clazzes.org: FTPSync.pl trac page, public subversion access to FTPSync.pl.
