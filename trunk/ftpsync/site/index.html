FTPSync.pl Homepage

General

FTPSync.pl is a PERL skript to synchronize a local directory tree and a remote FTP directory tree.

FTPSync.pl is also 's first open source project and is published under the GPL (GNU Public License).

FTPSync.pl is managed using the services of at ftpsync.sourceforge.net (project homepage) respectively sourceforge.net/projects/ftpsync/ ("internal" project page).

As of 2009-04-15 FTPSync.pl is under subversion control, the public SVN URL is https://ftpsync.svn.sourceforge.net/svnroot/ftpsync
